# DSA — Data Structures & Algorithms

A personal collection of Data Structures and Algorithms implementations across multiple programming languages.

## Languages

| Language   | Folder       |
|------------|--------------|
| JavaScript | `/js`        |
| Java       | `/java`      |

| Python     | `/python`    |

```
dsa/
```
